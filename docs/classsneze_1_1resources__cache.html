Detailed Description

template<typename Type, typename... Args>
class sneze::resources_cache< Type, Args >

Helper class to cache resources.

This class is used to cache resources, so that they are not loaded more than once, when unloading a resource the count is decreased, and when it reaches 0 the resource is destroyed

Note
resources must inherit from sneze::resource
Template Parameters
Typethe type of the resource
Argsthe types of the arguments to be passed to the init method of the resource
