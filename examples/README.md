### All examples will

- run in a window that cam be resized.
- window size and screen monitor will be saved on closing.
- can toggle between fullscreen and windowed mode by pressing ALT+ENTER.
- will close when the user presses the ESC key.
- handle window resizing and content scaling.

### List of examples

- [Hello](hello) : A simple example that shows how to create a window and add some text to it.
- [sandbox](sandbox) : A sandbox use internally by testing new features.
